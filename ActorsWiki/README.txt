ActorsWiki - приложение для управления актерами и фильмами
Это простое Flask-приложение для управления информацией об актерах и фильмах. Оно позволяет:
Просматривать список актеров,
Добавлять новых актеров,
Редактировать информацию об актерах,
включая добавление фильмов, Искать актеров и фильмы.

Технологии:
Flask: Микрофреймворк для создания веб-приложений на Python.

SQLAlchemy: ORM для работы с базой данных.

SQLite: Легковесная база данных.

HTML/CSS: Для создания пользовательского интерфейса.


Запуск:
запустить Project.py
после этого перейти по http://127.0.0.1:5000

Использование:
Главная страница
    На главной странице отображается список всех актеров.

    Вы можете искать актеров по имени с помощью поля поиска.

    Чтобы добавить нового актера, нажмите кнопку "Добавить актера".

Добавление актера
    На странице добавления актера введите имя и биографию.

    Нажмите кнопку "Добавить", чтобы сохранить актера.

Редактирование актера
    На странице актера нажмите кнопку "Редактировать".

    Вы можете изменить имя, биографию и добавить фильмы, связанные с актером.

    Используйте поле поиска, чтобы найти фильмы по названию.

Поиск актеров и фильмов
    На главной странице введите имя актера в поле поиска и нажмите "Найти".

    При редактировании актера введите название фильма в поле поиска, чтобы найти и добавить фильмы.
